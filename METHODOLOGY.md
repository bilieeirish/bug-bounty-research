# Bug Bounty Methodology

## Phase 1: Reconnaissance
1. Identify target protocol
2. Map attack surface
3. Review documentation
4. Check previous audits

## Phase 2: Analysis
1. Static analysis (Slither, Mythril)
2. Manual code review
3. Logic flow analysis
4. Economic attack vectors

## Phase 3: Exploitation
1. Proof of Concept development
2. Foundry test creation
3. Impact assessment
4. Risk evaluation

## Phase 4: Disclosure
1. Write detailed report
2. Submit to Immunefi
3. Cooperate with team
4. Responsible disclosure
